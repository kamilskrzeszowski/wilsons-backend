Recipe Library

Every recipe for the factory you're viewing. Recipes are created and changed in Recipe Costing and appear here automatically. Click any recipe for its full ingredient list; plan production on the Production Planner tab; print batch sheets on Mixing Sheets.

Production Planner

Drag a recipe onto a day (on a phone, use each day's + add… menu), then set how much to make and split it across bag sizes. Generate a picking sheet and pack plan per day.

Ingredients Database

The master list every recipe is built from. Rename or amend an ingredient here and it updates across all recipes automatically. To swap one ingredient for another inside a recipe, use the dropdowns on Edit Recipes.

Backup & Restore

By default all data lives in this browser. Turn on Online sharing below to make every PC share one live copy. Either way, export a backup file now and then and keep it safe.

🌐 Online sharing (everyone sees the same data)

No server or Azure needed. The shared data lives in a small free online store. Set it up once, then open the same file on every PC.

  1. Go to jsonbin.io → Sign up (free). On the dashboard open API Keys and copy your Master Key.
  2. Paste it below and click Create shared store (first PC only). Then enter the same Key + Store ID on the other PCs and click Connect.
Once connected, click this to get a copy with the key built in — upload that copy to your website host and anyone who opens the link is connected automatically (no typing). Keep that link private to your team.

The data sits on the free jsonbin service (fine for internal recipes; not as locked-down as your own systems). Your edit password still controls who can change things. Keep downloading a backup file as your own copy.

Recipe-edit password

Changing or deleting recipes and ingredients requires this password. Viewing, production planning and printing stay open for everyone. Use the lock button in the top-right to unlock or re-lock.

Note: this is a simple deterrent stored in this browser, not bank-grade security — it stops casual edits but a determined technical user could bypass it.
